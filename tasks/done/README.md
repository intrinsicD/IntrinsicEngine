# Done Tasks

Completed tasks should be moved here with completion metadata.

Each done task should include:

- Completion date.
- Commit hash and/or PR reference.
- Any follow-up tasks created from deferred scope.

Retirement narratives (the summary blocks formerly kept in
`tasks/active/README.md`) are appended newest-first to
[`RETIREMENT-LOG.md`](RETIREMENT-LOG.md); the log is append-only.

This directory holds only recent retirements. Older retired tasks are
periodically swept to [`tasks/archive/`](../archive/README.md), where their
IDs stay authoritative for dependency resolution but the files are frozen
read-only history.
